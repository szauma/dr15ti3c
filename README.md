   # Wlasne repozytorium kodu.

```php
use My\Super\Library\Class;
 
$data = [[1, 3], [1, 4], [2, 4], [3, 1], [4, 1], [4, 2]];
 
$object = new Class(1, 2, 3)
$object->makeAwesome($data);
```
